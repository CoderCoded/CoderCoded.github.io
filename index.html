We do full stack, and more.

Our design and implementation expertise covers following areas:

UX and UI

We create HTML5 based, mobile first responsive user interfaces using the latest Web technologies and frameworks like React and AngularJS. When HTML and CSS is not efficient enough, we bring Pixi.js into play.

Cloud Services

Our cloud service solutions are built on top of Node.js. We can rapidly prototype different approaches, and turn the best into scalable production services. Components vary according to customers needs.

Pepper - The Humanoid Robot

As the first Finnish member of ​SoftBank Robotics Pepper Partners Program, we create cutting edge robotics applications for the most advanced humanoid robot available in the market.

Interactive Installations

We create stunning interactive installations using cameras, video projectors, touch screens and other modern and experimental user interface technologies.

We are a team of three highly experienced MSc level programming professionals and enthusiastic hackers.
